JAVASCRIPT -interativo

tag: <comando>

Referenciando as TAGS no CSS: Todos comando devem ser feito entra os {} colocando sempre ; no final do código

1.Pelo id que é a indentificação única desse elemento
Ex:
No html:
<p id="inicio"></p>

No CSS:
#inicio{

2.Pela classe que é a indentificação em grupo de elemento por sua caracteristica/função.
Ex:
No html:
<h1 class="titulo"></h1>

No CSS:
.titulo{
font-size: 20px;
color:pink;
}

3.Pela própria TAG
Ex:
No html:
<p>Hoje...</p>

No CSS:
p{
font-size: 14;
font-family: 'Arial';
}
